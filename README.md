# CODTECH-TASK-1
Overview of Basic Calculator Program
This Python program implements a simple, text-based calculator that allows users to perform basic arithmetic operations: addition, subtraction, multiplication, and division. The user is prompted to input two numbers and select the desired operation. The program handles the following operations:

Addition (+): Adds two numbers.
Subtraction (-): Subtracts the second number from the first.
Multiplication (*): Multiplies two numbers.
Division (/): Divides the first number by the second, with a check to prevent division by zero.
The program is structured with four functions for the arithmetic operations (add, subtract, multiply, divide), and the main function, calculator(), coordinates user input, validates the operation, and displays the result.

Features:
User-friendly interface with clear prompts.
Error handling for division by zero.
Flexible input for any valid floating-point numbers.
This simple calculator can be easily extended to include more advanced operations or integrated into larger projects.
